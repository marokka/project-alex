"git init" - иницилизировать репозиторий
"git config" --local user.name "Alexander" - прописываем локальное имя для работы с репазиторием
"git config" --local user.email example@gmail.com - прописываем email
Если хотим глобально --global
"git status" - проверить статус репазитория 
"git add -A" - git All - добавить все файлы в репазиторий, моэно на примую обращаться к файлу, или написать *.тип файла и добавить только 
файлы которые имеют определенный тип данных
"git commit -a -m"my commit"" - коментируем свой добааленные файлы -а обозначет что берем все файлы -m - это сообщение
оптсываем что сделали
